# **App Name**: AVA Transfer

## Core Features:

- Transfer Form: A form to input transfer details like recipient name, bank, account number, and amount.
- Data Validation: Client-side validation of the input data, ensuring all fields are filled and of the correct format.
- Confirmation Screen: A confirmation screen to review the entered details before initiating the transfer.
- Transfer Simulation: Simulate the money transfer with a loading state and a delay, mimicking a backend process.
- Success Screen: Display a success message with transfer details upon successful simulation of the transfer.
- Receipt Download: Generate and download a PDF receipt of the transfer using jsPDF, with key details and a timestamp.

## Style Guidelines:

- Primary color: Dark navy blue (#1E293B) to evoke trust and stability.
- Background color: Very light grey (#F9FAFB), almost white, to give a clean and modern feel.
- Accent color: Deep sky blue (#0EA5E9) to highlight key actions and elements.
- Font: 'Inter' for body and headlines, a versatile sans-serif for a modern, clean, and professional look.
- Use minimalist, geometric icons to represent different actions and categories.
- Employ a clean, grid-based layout for easy readability and a professional look.
- Subtle transitions and animations to enhance user experience.
